# Image Processing

Basic examples of image processing functions in Java.

## implemented Functions:

```java
* toMediumGrayScale (); // Turns the image into gray

* toWeightedGrayScale (); // Turns the image into gray with weighted colors

* thresholding (int limit); // Threshold the image (turns it black and white only) 

* toNegative (); // Invert the image colors

* sumwith (String path); // Sums the image to another image

* sumWith (String path, double p1) // Sums the image to another using p1 as the percentage of the original image

* salientPoint (); // Salient point detector

* roberts (); // Roberts edge detection algorithm

* sobel (); // Sobel edge detection algorithm

* robinson (); // Sobel edge detection algorithm

* dilate (int[][] kernel) // dilate the image using the given kernel

* erode (int[][] kernel) // erode the image using the given kernel
```
